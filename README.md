# A3toA4Bookleter（A3小冊子スキャン→A4連番ページ一括変換ツール）

小学生の塾プリントなどでよくある、「A3サイズの用紙を半分に折って、中央をホチキス留めした小冊子」を、そのままA3スキャナーで読み込んだPDFファイルを処理するツールです。

裁断せずに見開きのままスキャンしたSearchable PDF（テキスト検索可能なPDF）を、プログラムが自動で「半分（A4サイズ）に分割」し、さらに**表紙から裏表紙までが正しい順番に揃ったSearchable PDF** に一括変換します。

---

## 🌟 特徴

* **文字も検索可能！**
  PDFの画像データを劣化させず、表示領域だけを半分にして並べ替えるため、スキャナーで読み取られた「テキストデータ（OCR）」がそのまま残ります。
* **面付け（ページ順）を自動判定！**
  表紙などの印字から「国語や漢字（右綴じ）」か「算数や理科（左綴じ）」かを自動で判断し、正しいページ順で結合します。
* **縦長・横長の給紙方向のブレも自動補正！**
  スキャナー特有の「横倒し」に読み込まれてしまったページも自動検知して、A4の正しい向き（縦向き）に補正して出力します。

---

## 🛠️ 準備（初回のみ）

このツールはMacに入っている「ターミナル」という黒い画面のアプリを使います。Python（プログラミング言語）を使いますが、**プログラミングの知識は一切不要**です！必要な設定もツールが自動で行います。

1. **Macの「ターミナル」アプリを開く**
   画面右上の虫眼鏡マーク（Spotlight検索）をクリックし、「ターミナル」と入力して開きます。

---

## 🚀 使い方

### Step 1: スキャンしたPDFを準備する
1. A3見開きモード（または両面スキャン）でスキャンしたPDFファイルを用意します。
2. このフォルダの中にある **`input`** フォルダの中に、そのPDFファイルを入れます。（複数ファイルを入れておけば一気に処理されます）

### Step 2: ターミナルでフォルダを読み込む
ターミナルの黒い画面に、文字を打ち込みます。

1. 半角英数字で `cd ` と打ち込みます。
2. Finderから、**いま見ているこの「A3toA4Bookleter」というフォルダそのもの** を、ターミナルの中にドラッグ＆ドロップします。
3. `cd /Users/あなたの名前/Desktop/xxxx/A3toA4Bookleter` のようにパスが自動入力されたら、**Enter（Return）キー**を押します。

### Step 3: プログラムを実行する
1. 続いて、ターミナルに以下をコピー＆ペーストして **Enterキー** を押します。

```bash
python3 convert.py
```

*(初めて実行するときは、PDFを処理するためのパッケージが自動でインストールされるため少しだけ文字が流れます)*

### Step 4: 完了！
ターミナルに「全ての処理が完了しました」と表示されたら作業完了です。

* **`output` フォルダ:** A4サイズになってページ順が揃った完成版のPDFが保存されます！
* **`input/done` フォルダ:** 処理が終わった元ファイル（A3のPDF）は、間違って二重に処理されないように自動でこちらに退避・移動されます。

---

## ⚠️ 注意点（うまく読み取れない場合）
1ページ目だけでなく全体が文字化けするような極端に不鮮明なスキャンの場合、自動での「国語（右綴じ）」の検出ができず、標準設定である「算数など（左綴じ）」として出力されてしまう場合があります。出力されたPDFを確認してみて、ページ順が逆でしたら、出力ファイルを破棄してご自身でプレビューアプリなどでページ順を入れ替えていただく必要があります。
